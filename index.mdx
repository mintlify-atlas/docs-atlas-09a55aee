## Quick Start

Get your Bancard payment gateway up and running in three simple steps:

<Steps>
  <Step title="Install the Module">
    Add the Bancard payment provider module to your Odoo instance
    
    ```python
    # Add to your Odoo addons directory
    git clone https://github.com/YamilTorricoVasquez/bancard
    ```
  </Step>
  
  <Step title="Configure Provider">
    Set up your Bancard API credentials in Odoo payment settings
    
    Navigate to **Website → Configuration → Payment Providers** and configure your public and private keys
  </Step>
  
  <Step title="Test Integration">
    Process a test transaction using the staging environment
    
    Use the staging URL: `https://vpos.infonet.com.py:8888`
  </Step>
</Steps>

## How It Works

The Bancard payment gateway integrates directly into your Odoo e-commerce checkout flow:

1. **Customer Checkout** — Customer proceeds to payment in your Odoo store
2. **Payment Initiation** — Odoo sends payment request to Bancard vPos
3. **Secure Payment** — Customer completes payment in Bancard's secure iframe
4. **Webhook Confirmation** — Bancard notifies your system of payment status
5. **Order Processing** — Odoo automatically confirms and processes the order
